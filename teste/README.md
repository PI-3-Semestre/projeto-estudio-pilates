# Bibliotecas Utilizadas
- email-validator faz validação de emails e confirma a existência do domínio: 
`poetry add email-validator`
`

- uuid gera identificadores únicos: `import uuid`

- Pillow Realiza tratamento de imagens: `poetry add Pillow`

- phonenumber Faz validação de números de telefone/celular: `poetry add phonenumbers`

django-cpf Faz validação de CPF, somente para django: `django-cpf`
